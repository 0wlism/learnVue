# learnVue

## 介绍

Vue.js源码学习笔记，记录了个人学习Vue.js源码的过程中的一些心得以及收获。

[vue-src](./vue-src)是我在学习的过程中为Vue.js源码加上一些注释，希望可以帮助有需要的同学更好地学习理解Vue.js源码。

可能会有理解存在偏差的地方，欢迎提issue指出，共同学习，共同进步。
